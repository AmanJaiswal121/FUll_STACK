Methods in java

It is a sub block of a class that is used for implemeting logic of an objects operations.

    NOTE : logic must be placed only inside a method, not directly at class level. If we place logic at class level compiler throws error.



01. With Initlization -

class Main{
int a,b;
int sum;
// here we can initlize the variable.
public int add (int a, int b){
sum = a + b;
// return type of method.
return sum;
}
public static void main(String[] args) {
Main r = new Main();
// here we give value tp our initlize variable.
System.out.println(r.add(10,4));
}
}

output : 14

Explaination -

    In this programm our global variables are not initalize at compile time. We give then values after creating object.
