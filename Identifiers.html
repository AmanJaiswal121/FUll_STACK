Identifiers

Any class name, object name, method name, variable name in java programm is known as identifiers which is for identification purpose only.


Rules in identifiers

  • a-z, A-Z, 0-9, _, $ only these are use as identifiers.

  • We cannot start identifier name with digit.

  • Limitation is not the boundary for name.

  • We cannot use reserve words (keywards) for the identification of identifier.

eg. - find all identifiers in this programm.

public class Test{
public static void main(String[] args){
System.out.println("Hello Java");
}
}

Output : Hello Java




In this programm 7 identifiers are their -

  • Test - class name.
  • main - method name
  • String - class name
  • args - variable name
  • System - class name
  • out - static variable of System class and object of print method.
  • print - method name.
