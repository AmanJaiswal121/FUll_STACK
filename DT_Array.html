Array

An array is a linear data structure which can be defined as a finite ordered set of homogenious elements that collects elements of same data type and store them in continous and adjacent memory locations.




The above image can be looked as a top-level view of a staircase where you are at the base of the staircase. Each element can be uniquely identified by their index in the array (in a similar way as you could identify your friends by the step on which they were on in the above example).

Array indexing start from 0 to (n - 1)

    n is a size of array.



Basic operation which perform in array -

  • Traverse : print all the array element one by one.

  • Insertion : add an element at he given index.

  • Deletion : deletes an element at the given index.

  • Search : searches an element using the given index or by the value.

  • Update : updates an element at the given index.

Elements in an array are sorted at contigous memory locations from which the first element is stored at the same smallest memory location.


Types of Array

One dimension/Single dimension array -

A one dimension arrray is the simplest form of array in which the elements are stored linearly and can assessed individually by specifing the index value of each element stored in array.

Two dimension/Multi dimension array -

Multidimensional Arrays can be defined in simple words as array of arrays. Data in multidimensional arrays are stored in tabular form (in row major order).

Size of multidimensional arrays: The total number of elements that can be stored in a multidimensional array can be calculated by multiplying the size of all the dimensions.

For example : The array int[][] x = new int[10][20] can store a total of (10*20) = 200 elements. Similarly, array int[][][] x = new int[5][10][20] can store a total of (5*10*20) = 1000 elements.
