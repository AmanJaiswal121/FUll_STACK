Continue & Break statement

The break and Continue statement are jump statement that are used to skip some statements inside the loop or terminate the loop immediatly without checking the test expression.

These statements are used inside any loops such as for, while, do-while.

Continue statement in java :

It is used to skip the current iteration of a loop. We can use continue statement inside any type of loops such as for, while, do-while.

Basically, if we want to run our loop, but at point (according to condition) we dont want to run only that point then we use this continue statement.

Continue statement using for loop -

class A{
public static void main(String[] args) {
for(int i = 0; i <= 5; i++) {
if(i == 3) {
System.out.println("End Hello only at 3");
here loop got break continue;
}
System.out.println("Hello at " + i);
}
}
}

Output -
  Hello at 0
  Hello at 1
  Hello at 2
  End Hello only at 3
  Hello at 4
  Hello at 5


Continue statement using while loop -

class A{
public static void main(String[] args) {
int i = 0;
while(i == 5) {
if(i == 3) {
i++;
System.out.println("Not print 3");
here loop got break continue;
}
System.out.println("Hello at " + i);
}
}
}

Output -
  Hello at 0
  Hello at 1
  Hello at 2
  End Hello only at 3
  Hello at 4
  Hello at 5



Break statement in java :

It is used to terminate the loop immediatly.

Basically, break statement are used in situation when we are not sure about the actual number of iteration for the loop or we want to terminate the loop based on condition.
